# Edge AI

This module performs machine health analysis using sensor measurements collected from industrial equipment.

The Edge AI engine detects abnormal behavior, estimates remaining useful life (RUL), and predicts potential motor failures.

---

## AI Tasks

- Anomaly Detection
- Motor Failure Prediction
- Remaining Useful Life Estimation

---

## Input Parameters

- Current
- Power
- Vibration (Future)
- Temperature (Future)
- Noise (Future)
