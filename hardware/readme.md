# Hardware

This folder contains the embedded hardware architecture of Electra.

The hardware platform continuously monitors industrial machine health by collecting electrical and environmental parameters and transmitting them to the Edge AI platform.

---

## Hardware Components

- ESP32
- INA219 Current Sensor
- ACS712 Current Sensor (Future)
- Vibration Sensor (Future)
- DHT Sensor (Future)
- Noise Sensor (Future)
- Relay Module (Emergency Shutdown)

---

## Responsibilities

- Current Measurement
- Power Monitoring
- Machine Health Monitoring
- Sensor Data Acquisition
- MQTT Communication
